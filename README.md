# xmlfmt

## Install

```Shell
% go get install code.jalapeno.red/xmlfmt
```

## Use

```Shell
% curl -s http://www.w3schools.com/xml/note.xml | xmlfmt
<note>
  <to>Tove</to>
  <from>Jani</from>
  <heading>Reminder</heading>
  <body>Don&#39;t forget me this weekend!</body>&#xA;&#x9;&#xA;&#x9;&#xA;&#x9;&#xA;&#x9;&#xA;
</note>
```
